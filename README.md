# 🏦 Bank Transaction System — SQL Project

## 📌 Overview
A relational database project simulating core banking operations 
including customer management, account handling, and transaction tracking.

## 🛠️ Tech Stack
- SQL (MySQL)

## 📂 Files
| File | Description |
|------|-------------|
| schema.sql | Database & table creation |
| sample_data.sql | Dummy data for testing |
| queries.sql | Analysis & MIS-style queries |

## 🔍 Key Queries Covered
- Customer balance summary
- Transaction history per customer
- Credit vs Debit analysis
- Monthly MIS report
- High-value transaction alerts
- Low-balance risk detection

## 💡 Concepts Used
- DDL & DML commands
- Primary & Foreign Keys
- JOINs (INNER JOIN)
- GROUP BY, HAVING
- CASE WHEN
- DATE functions
- Aggregate functions (SUM, COUNT)
